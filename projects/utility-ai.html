Utility AI — Autonomous Agent (C++)

Hybrid FSM + Utility AI system for a zombie survival game. The agent evaluates context every frame to explore, collect items, flee, or engage enemies.

C++AISystemsSolo

2025

Demo

Goal

Build an agent that survives autonomously using a robust decision system. I combined a high-level FSM (Explore / Attack / Flee) with a per-state Utility AI which scores actions and executes the best one each frame.

My Role

  • Designed the Utility AI framework (scoring & selection loop)
  • Integrated FSM states and per-state action sets
  • Implemented item handling, survival priorities, and steering
  • Built lightweight debug UI & tunable parameters

Tech Stack

  • C++ (custom framework), Visual Studio
  • FSM, Utility AI, Steering (seek/flee)
  • Git for version control

Highlights

  • Context-aware action scoring (health, ammo, distance, threats)
  • Exploration vs. item seeking vs. combat prioritization
  • Partial observability + memory (simple blackboard)
  • Extensible actions (plug-and-play interface)
